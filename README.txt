Our project was supposed to be a portal for employees for Atom bank.

All out ideas and plans you can find on
https://www.figma.com/board/YzNej2zCh0FXw0c2xLIIHg/Atom-Portal?node-id=0-1&node-type=canvas&t=HQyAaJCM0GNTTuDY-0

If you want to start a python file and see the python concept, execute A_MainControl.py file

If you want to see the concept of a timetable, execute timetable.py file

You might need to install additional libraries like
tkinter
hashlib
matplotlib
pygame
etc.
